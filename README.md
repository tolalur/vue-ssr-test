# vue-ssr-test

Фронт: vue2, vue-router, composition-api, typescript  
Бек BFF: Nest.js, typescript

## Backend
Установка
```bash
$ npm ci
```

Разработка:
### development
```bash
$ npm run start
```
### watch mode
```bash
$ npm run start:dev
```
### production mode
```bash
$ npm run start:prod
```

## Front
Установка
```bash
$ npm ci
```

### запуск ssr и фронта
```bash
$ npm run dev
```

### production mode
```bash
$ npm run build
```
